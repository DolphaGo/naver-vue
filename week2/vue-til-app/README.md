# vue-til-app

## Project setup
```
npm install
```

### 프로젝트 생성
```
vue create todo-app
```


### Compiles and hot-reloads for development : 프로젝트 실행 및 개발 웹펙 데브 서버
```
npm run serve
```

### Compiles and minifies for production  : 서버 배포(웹팩) (변환하는 동작 (빌드 /변환/ 번들링, 다 같은 말) )
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
